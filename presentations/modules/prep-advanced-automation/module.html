Pre-Workshop Instructions

Please perform the following steps before the workshop. Some of these steps take a few minutes and you may have issues specific to your machine that we can address in advance. Email me if you get stuck.

STEP 1: Install the latest Cytoscape

  • Download Cytoscape v3.6.0
  • Open and follow installation steps

Pre-Workshop Instructions

STEP 2: Install R and RStudio

  • You will need R version 3.3.3 or greater
  • You will need RStudio 1.0.143 or greater

Pre-Workshop Instructions

STEP 3: Download workshop files

  • OPTION 1: If you are familiar with git, then fork and clone our repo:
    • Click the Fork button
    • Clone from your repo, e.g.:
    git clone git@github.com:USERNAME/cytoscape-automation.git
  • OPTION 2: Or you can download this file and unzip it: github.com/cytoscape/cytoscape-automation/archive/master.zip

Pre-Workshop Instructions

STEP 4: Load workshop files in RStudio

  • Launch RStudio
  • Start a new project: File>New Project...
  • Choose Existing Directory and naviate to the directory: cytoscape-automation/for-scripters/R

Pre-Workshop Instructions

STEP 5: Run the check scripts

  • Run check-library-installation.R. Read the comments in the script for more information.
  • Run check-cytoscape-connection.R. Note that you will need to have Cytoscape 3.6.0 running and the STRING app installed:
    • STRING app

Pre-Workshop Instructions

That's it! If the check scripts ran without printing "Warnings" and you ended up with an animated 4-node network in Cytoscape, then you are ready for the workshop.

If you ran into any issues along the way, please email me ahead of time.

Thanks!
